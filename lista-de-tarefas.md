## Projeto 01 - Análise de vendas de uma loja de E-commerce

Para executar esse projeto dividi ele em pequenas etapas pra organizar tudo o que devia ser feito, visualizar o progresso e manter o controle
sobre o desenvolvimento, do início ao fim.

### Tarefas:

- [x] Criar uma função pra gerar um dataset de maneira aleatória, contendo colunas de cliente, produto, preço, categoria, quantidade e data.

- [x] Realizar as transformações no dataset, de forma a deixá-lo limpo e organizado
    - [x] Remoção de valores nulos
    - [x] Organização das datas
    - [x] Remoção de duplicatas se houver
    - [x] Padronização das datas conforme a norma nacional NBR 5892:2019 "dd/mm/yyyy”

- [x] Salvar o dataset gerado em csv na camada silver.
